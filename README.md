SPEI Fraud Detection

Machine learning model for detecting fraudulent transactions on Mexico's
real-time payments rail (SPEI), built on synthetic data calibrated to
Banxico's published aggregate statistics.

Stack

- Python 3.11
- pandas, pytest
- Streamlit (planned, Week 8)

Setup


python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
